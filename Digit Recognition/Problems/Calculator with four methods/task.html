Calculator with four methods Open on JetBrains Academy

In a template below you have a simple calculator inside the main method that subtracts, sums, divides and multiplies two numbers. What if we decompose it in four separate methods that perform specific actions?

Let's create four methods for each operation of the calculator: subtractTwoNumbers(long a, long b) for subtraction, sumTwoNumbers(long a, long b) for adding, divideTwoNumbers(long a, long b) for division and multiplyTwoNumbers(long a, long b) for multiplication. Use the provided template below.


Note that you can't divide by zero. In case your second argument is zero, you should return the first argument of the division.


Sample Input:
1 + 2
Sample Output:
3
